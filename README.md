# BoxNestGroup

Box の グループをネスト(入れ子)管理するWPFアプリ

## 1.概要

｢現在、学習しながら作成中｣

Box の グループをネスト(入れ子)管理して、グループに設定したユーザーを上位のグループを含めて設定できるアプリ

(を目指しています)

こんな感じです。

｢グループ A｣
├｢グループ B｣
├｢グループ C｣
:└｢グループ D｣

｢グループD｣にユーザー設定すると、上位の｢グループC｣｢グループA｣も一緒に登録する
